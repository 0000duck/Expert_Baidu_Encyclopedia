表面单分子器件相关的理论计算研究，氧化物催化的甲烷高效利用和光催化产氢的理论计算研究，新型钒氧化物功能材料的理论计算研究以及其他新型材料的理论设计和性能预测。单分子器件方面主要包括：基于数学原理，编程实现了扫描隧道显微镜（STM）的快速模拟（J. Phys. Chem. A 2014, 118, 8953）；运用电子结构分析和自编程数值模拟，提出利用电极和分子的轨道之间的空间对称性匹配来调控单分子的输运性质的新机制，解释了STM实验中观察到的负微分电阻现象（Phys. Rev. Lett.2007, 99, 146803）、针尖对分子-衬底杂化态选择性探测现象（J. Phys. Chem. C 2008, 112, 15603）和钴酞菁分子在金表面台阶处的近藤效应（J. Chem. Phys. 2008, 128, 234705），并从理论角度研究金表面上不同金属酞菁分子电子和磁学性质、预测发生近藤共振的可能性（J. Phys. Chem. C 2008, 112, 13650）、以及范德华力对体系性质的影响（J. Phys. Chem. C. 2014, 118, 27843）。甲烷、氢气等新能源催化方面主要包括：通过基本的物理学原理设计复合二氧化钛材料提高光催化产氢效率（ACS Catalysis 2015, 5, 1708）；通过计算构建了甲烷碳氢键在CeO2(111)和Pt掺杂CeO2(111)表面的活化路径，揭示了甲烷在Pt掺杂CeO2(111)表面催化氧化的微观机制（J. Catal. 2010, 273, 125）；通过模型构建，确定反应中间产物，协助实验建立CO2在Ru掺杂CeO2表面加氢还原的反应模型（J. Catal. 2011, 278, 297）；提出以能量这一最基本物理量为判据选取钛氧化物体系U值的方法（J. Phys. Chem. C 2011, 115, 5841）；以氧缺陷生成能为参数，通过计算发现参杂氧化物体系中杂质的局域效应—键合效应、广义效应—价态效应和CeO2体系的自掺杂效应（J. Phys. Chem. C 2011, 115, 17898）；通过计算在CeO2体系发现了低价金属参杂氧化物中的化学补偿效应，并在其他氧化物体系获得了验证（J. Phys. Chem. C 2011, 115, 3065）;卤素在CeO2(111)表面吸附的计算研究，发现卤素在氧化物表面能够以电子给体和电子受体两种形式存在，并且存在很强的相互作用，具有潜在的催化应用（J. Phys. Chem. C 2012, 116, 6664）。新型钒氧化物功能材料的理论计算研究以及其他新型材料的理论设计方面主要包括：同实验合作，利用能带调制设计VO2/V2O5同素异质结红外探测材料（Adv. Funct. Mater. 2014, 24, 1821）；通过计算揭示氮化钒纳米颗粒各向异性导电性的物理本质（Nano Reserch 2015, 8, 193）；计算设计能带可调的碳纳米材料（J. Phys.Chem. Lett. 2012, 3, 3373）和金属有机层状材料（J. Phys. Chem. C 2014,118, 5593）。通过计算解释钒氧化物（Chem. Commun. 2008, 3891）和铜锡硫材料（Appl. Phys. Lett. 2007, 91, 143104）的导电机制。
